
Image assets guide
==================
Put your images into: assets/img/

Required/used filenames (you can rename, but update HTML if you do):
- heart.png                -> heart icon on CREATED page
- shape-yellow.png         -> left big yellow bowl (HOME)
- shape-green.png          -> left green clover (HOME)
- shape-pink.png           -> left pink star (HOME)
- shape-purple.png         -> right purple diamond (HOME)
- shape-orange.png         -> right orange L (HOME)
- shape-blue.png           -> right blue gear/sunburst (HOME)
- shape-pink2.png          -> right pink block (HOME)

All images recommended transparent PNG, ~2x display size for retina.
